Technical Manual

Sphinx
------

Here are some Sphinx commands we've used to create this documentation:
- `python -m sphinx.cmd.quickstart`: Generates a new Sphinx project.
- `python -m sphinx.cmd.build -b html .\source .\build\html`: Builds HTML documentation from the source files.

Check out the official `Sphinx documentation <https://www.sphinx-doc.org/>`_ for more details.
